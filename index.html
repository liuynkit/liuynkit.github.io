Meaning of the columns in the table below:

  1. The noisy audio input to the model. It's generated by summing the clean audio with interference audio from another speaker. Adding noise and reverberate are sometime used
  2. The output from the enhancing model.
  3. The reference audio from which we extract the x-vector. The x-vector is another input to the enhancing model. This audio comes from the same speaker as the clean audio.
  4. The clean audio, which is the ground truth.
